# AC - 减重小程序 · 用户端

> 版本: V1.0
> 日期: 2026-06-05
> 范围: 用户端微信小程序（登录引导 / 首页 / 打卡 / 饮食记录 / 数据分析 / 营养师 / 我的 / 暂不减重分支）
> 原型: `6.小程序原型/standalone/`
> - `01-登录引导.html` `02-首页.html` `03-数据分析.html` `04-饮食记录.html` `05-消息.html`(营养师) `06-我的.html`
> - 副本: `02-首页-暂不减重.html`（暂不减重·无方案态）
> 底部导航: 首页 / 数据 / ➕(添加记录) / 营养师 / 我的

字段口径说明:
- **餐别**: 早餐 / 早加餐 / 午餐 / 午加餐 / 晚餐 / 晚加餐（加餐细分为三类）。
- **打卡项（7类）**: 饮食 / 体重 / 三围 / 尿酮 / 排便 / 生理期 / 睡眠与作息 / 运动（饮食以"饮食热量"卡呈现）。
- **达标率**: 单项实际完成 / 方案目标，分饮食 / 运动 / 饮水 / 睡眠 四项。
- **方案**: 用户首次评测（减重目标 + 健康数据）生成的量化方案；营养师后台可调整。
- **打卡状态色点（日历）**: 合适(绿) / 吃多了(红) / 吃少了(琥珀) / 未打卡(无)。
- **还可吃** = 推荐预算 − 饮食摄入 + 运动消耗×0.9。
- **无方案态**: 用户选"暂不减重"后的首页状态，体重卡仅展示当前 + 健康上下限，无减重方案。
