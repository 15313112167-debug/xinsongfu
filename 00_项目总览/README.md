## 一、项目概述

| 项目名称 | 新松福健康管理小程序 |
|---------|-------------------|
| 项目类型 | 减重健康管理平台（小程序 + 后台系统） |
| 一期范围 | 用户端（微信小程序） + 营养师后台（PC） + 管理后台（PC） |
| 当前阶段 | 需求细化与原型设计阶段 |
| 目标上线 | 2026年6月底 |

### 核心功能模块

| 端 | 核心模块 | 状态 |
|---|---------|------|
| **用户端** | 登录引导 / 首页打卡 / 饮食记录 / 数据分析 / 营养师 / 我的 | 🟡 设计中 |
| **营养师后台** | 用户管理 / 评餐管理 / 方案制定 / 数据统计 / 异常预警 | 🟡 设计中 |
| **管理后台** | 营养师管理 / 全量用户管理 / 角色权限 / BI报表 | 🟡 设计中 |
