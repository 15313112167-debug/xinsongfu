减重类小程序里，“特殊疾病风险提示”非常重要，因为很多用户并不适合：

- 极低热量饮食
- 生酮饮食
- 高强度运动
- 快速减重

如果没有做风险提示，后续容易出现：

- 医疗风险
- 客诉
- 合规问题
- 平台责任问题

建议你把疾病分成：

# **一、必须强风险提示（高危）**

这些建议：

- 强提醒
- 限制部分功能
- 建议医生指导
- 必要时禁止方案自动生成

------

## **1. 糖尿病（尤其胰岛素使用者）**

### **风险原因**

减重、低碳、生酮容易导致：

- 低血糖
- 酮症酸中毒
- 药物剂量失衡

------

### **建议提示**

您当前存在糖尿病相关情况，请在医生或营养师指导下进行减重。

------

### **后台规则建议**

| **规则**         | **建议** |
| ---------------- | -------- |
| 禁止极低碳方案   | √        |
| 禁止快速减重     | √        |
| 增加血糖监测提醒 | √        |
| 高风险用户标签   | √        |

------

## **2. 高血压 / 心血管疾病**

包括：

- 冠心病
- 心衰
- 心律失常
- 心梗史

------

### **风险**

高强度运动可能诱发：

- 心脏负荷异常
- 血压波动

------

### **后台建议**

| **功能**         | **建议** |
| ---------------- | -------- |
| 限制运动强度     | √        |
| 不推荐 HIIT      | √        |
| 控制每日热量缺口 | √        |

------

## **3. 肾病（重点）**

包括：

- 肾功能不全
- 肾炎
- 肾结石高风险

------

### **风险**

高蛋白减重容易：

- 增加肾负担
- 加重肾功能损伤

------

### **后台规则**

| **功能**       | **建议** |
| -------------- | -------- |
| 禁止高蛋白方案 | √        |
| 限制生酮       | √        |
| 强制人工审核   | 推荐     |

------

## **4. 肝病**

包括：

- 脂肪肝
- 肝炎
- 肝功能异常

------

### **风险**

极端减重可能：

- 加重肝损伤
- 引发代谢异常

------

### **建议**

| **功能**     | **建议** |
| ------------ | -------- |
| 限制快速减重 | √        |
| 增加饮食提醒 | √        |

------

## **5. 痛风 / 高尿酸**

这个减重场景很常见。

------

### **风险**

快速减重、生酮饮食容易：

- 尿酸升高
- 诱发痛风

------

### **后台规则建议**

| **功能**     | **建议** |
| ------------ | -------- |
| 限制生酮方案 | √        |
| 限制高蛋白   | √        |
| 增加饮水提醒 | √        |

------

## **6. 进食障碍（重点风控）**

包括：

- 暴食症
- 厌食症
- 神经性贪食

------

### **风险**

减重产品可能：

- 加重心理问题
- 引发危险行为

------

### **建议**

| **功能**       | **建议** |
| -------------- | -------- |
| 不允许激进减重 | √        |
| 增加心理提示   | √        |
| 必要时限制使用 | 推荐     |

------

## **7. 孕期 / 哺乳期**

这是必须识别的。

------

### **风险**

普通减重方案可能影响：

- 胎儿
- 营养摄入
- 泌乳

------

### **建议**

| **功能**         | **建议** |
| ---------------- | -------- |
| 禁止减重方案     | √        |
| 转“体重管理模式” | 推荐     |
| 特殊营养建议     | √        |

------

# **二、中风险疾病（需提示）**

这些通常不需要禁用，但需要“方案调整”。

------

## **8. 甲状腺疾病**

包括：

- 甲减
- 甲亢

------

### **风险**

影响：

- 基础代谢
- 体重波动

------

### **建议**

| **功能**     | **建议** |
| ------------ | -------- |
| 调整减重预期 | √        |
| 提示代谢异常 | √        |

------

## **9. 多囊卵巢综合征（PCOS）**

女性减重高频场景。

------

### **建议**

| **功能**     | **建议** |
| ------------ | -------- |
| 单独标签     | √        |
| 推荐控糖方案 | √        |
| 增加周期跟踪 | √        |

------

## **10. 睡眠呼吸暂停**

通常伴随肥胖。

------

### **风险**

运动耐受差。

------

### **建议**

- 睡眠监测
- 控制运动强度

------

## **11. 骨关节疾病**

包括：

- 膝关节炎
- 腰椎问题

------

### **建议**

| **功能**       | **建议** |
| -------------- | -------- |
| 避免跳跃运动   | √        |
| 推荐低冲击运动 | √        |

------

| 焦虑/抑郁 | 情绪饮食分析 |

------

# **四、建议你和业务重点沟通的“风险等级”**

建议后台做：

| **风险等级** | **处理方式** |
| ------------ | ------------ |
| 低风险       | 正常方案     |
| 中风险       | 调整方案     |
| 高风险       | 强提示       |
| 极高风险     | 人工审核     |

------

# **五、最容易被遗漏的“高风险用户”**

这个你一定要问业务。

------

## **1. BMI 过低用户**

例如：

- BMI < 18.5

不应该继续减重。

------

## **2. 未成年人**

建议：

| **年龄** | **建议** |
| -------- | -------- |
| < 12岁   | 禁止减重 |
| 12–18岁  | 家长同意 |

------

## **3. 老年人**

风险：

- 肌肉流失
- 营养不良

------

## **4. 极端减重目标用户**

例如：

- 一个月减20斤

建议：

- 风险提示
- 限制目标

------
