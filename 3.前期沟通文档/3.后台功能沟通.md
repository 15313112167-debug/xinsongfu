
# **一、用户基础信息与评测规则**

## **1. 用户基础信息采集规则**

需要确认：

| **功能点** | **需要沟通的问题**        |
| ------- | ------------------ |
| 性别      | 是否仅支持男女？是否预留其他性别   |
| 身高/体重   | 数值范围是多少？最小最大值      |
| BMI计算   | BMI公式是否统一采用 WHO 标准 |
| BMI分层   | 偏瘦/正常/超重/肥胖 的区间定义  |
| 年龄建议    | 不同年龄段是否有不同减重建议     |
| 基础信息修改  | 用户修改后是否影响历史分析      |
| 数据更新频率  | 修改后是否实时刷新全部指标      |
