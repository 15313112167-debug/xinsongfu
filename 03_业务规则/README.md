## 数据层级设计（来自采集数据整理.md）

```
基础档案 → 健康数据 → 行为数据 → 过程数据 → 风险数据 → 运营数据
```

| 层级 | 核心字段 |
|-----|---------|
| 基础档案 | 性别/年龄/身高/体重/目标体重/目标周期 |
| 健康数据 | BMI/体脂率/腰围/臀围/疾病史/用药情况 |
| 行为数据 | 餐饮记录/饮水量/运动类型/睡眠时长 |
| 过程数据 | 连续打卡天数/达标率/方案执行情况 |
| 风险数据 | 尿酮等级/异常预警/平台期判断 |
| 运营数据 | 满意度/减重信心/用户画像标签 |
