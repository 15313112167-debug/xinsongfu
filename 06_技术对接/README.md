## 技术架构说明

### 系统架构

```
用户端(微信小程序)
    ↓ API
服务端
    ↓ 数据库
MySQL数据库
    ↓ 缓存
Redis缓存
```

### 三端技术方案

| 端 | 技术栈 | 说明 |
|---|--------|------|
| 用户端 | 微信小程序原生 | 微信生态 |
| 营养师后台 | Vue.js + Element UI | PC端Web |
| 管理后台 | Vue.js + Element UI | 与营养师后台同一项目 |
